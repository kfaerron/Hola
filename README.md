## Controles
Para este juego de Tetris se ha elegido usar la disposición de controles "WASD" y la tecla "R".  
### **Función de cada tecla:**    
-**W:** No tiene función  
       -**A:** Mover un cuadro a la izquierda  
-**S:** Hace que la pieza caiga más rapido  
-**D:** Mover un cuadro a la derecha  
-**R:** Hacer una rotación de la pieza
